This project uses Python 3.9.4 and it is required to properly run the script as the libraries depend on it.  

If using Visual Studio Code:  
    &emsp;- Once Python 3.9.4 is installed, run command: "Python: Select Interpreter" and select the correct version.  
    &emsp;- To execute the script select "Run Python Code".
